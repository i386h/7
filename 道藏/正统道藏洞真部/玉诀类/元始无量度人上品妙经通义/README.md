元始无量度人上品妙经通义
　　经名：元始无量度人上品妙经通义。明四十三代天师张宇初撰。四卷。底本出处：《正统这藏》洞真部玉诀类。
　　元始无量度人上品妙经通义卷之一
　　正一嗣教道合无为阐祖光范真人领道教事四十三代天师张宇初注
　　太极妙化神灵混洞赤文之图
　　河图象数之图
　　天一地二、天三地四、天五地六、天七地八、天九地十，天数五地数五，五位相得而各有合。天数二十有五，地数三十。凡天地之数，五十有五，此所以成变化而行鬼神也。五方除生成本三数，则龙从火出，虎向水生，而母隐子胎，子归母腹。皆金丹七返，朱砂九还、金液之妙也。故与经中十四之义无殊为。
　　雷霆一窍之图
　　《抱朴子》曰：玄牝一家者，非、心、肾、鼻、口、肝、肺、谷道、膀胱、泥丸。知此一穷，则冬至火候，药物沐浴，脱胎结胎，皆在此矣。此窍即雷霆之窍，得之则雷露摄机，不必外求也。至人冲举，所用在此。道法感通，特余事耳。
　　雷霆枢机互用之图
　　雷 雷为天之枢，其性善、其主生、其体阳、其象阴、故甲丙庚壬为阳雷。
　　霆 霆为地之机，其性恶、其主杀、其体阴、其象阳、故乙己丁辛为阴霆。
　　雷霆互用
　　乾坤同体，日月合璧。动而霹露。

　　太上洞玄灵宝无量度人上品妙经
　　太者，大也。上者，尊也。无上之上，是为极尊。洞玄者，品秩之称。经有三洞：洞真，元始主之；洞玄，灵宝主之；洞神，道德主之。此经本元始所宣，而道君纪录，故标《洞玄灵宝》。而正经止云《元始无量度人上品妙经》，所以别前、后序。道言乃道君之言也。合而命曰《太上洞玄灵宝》，以由道君承元始之旨，无量度人，即一卷之目。三洞之中三十六部，此经为首，故曰上品。妙者，至精至微，玄奥难极也。经者，径也，日用修真、共由之道也。内而修之曰丹，即先天真一之祖炁也。上无复祖，惟道至尊。洞者，空洞明彻；玄者，湛寂幽微。灵乃真性长存，宝乃元命流行。能修之於身，则能度人，所度无穷，非可限量也。夫证上乘之道，不离乎自心。此心即道即经，是以虚灵无碍，而度人无量矣。
　　道言：昔於始青天中碧落空歌大浮黎土，受元始度人无量上品。
　　道言者，太上玉晨道君之言也。此前序之题，乃道君述昔受经始末，纪载成篇之因。始青者，东方九炁青天也。乃神霄九天之一。此天梵炁所凝，碧霞廓落，故曰碧落。碧霞之炁，凝为琼林瑶树，灵风鼓荡，自成空洞之歌，是曰空歌。大浮黎土即东极国名。道君谓我昔於始青天浮黎土，受元始天尊妙经，广度天人，是为无量上品。夫土，中宫也；玄，关也。其数居五，为天地万物之祖，故百昌生乎土，终乎土。此吾玄关，即宝珠也。元神，即元始也。万有生化无穷，皆由乎是也。始青，木之象也。火，龙也。震数三，乾初索而生长男，犹春居四序之首，万物以之而发生也。然一初生水，水生木；二乃生火，火由木生，此五行循环，生生无穷之理，可谓着矣。
　　元始天尊当说是经，周回十过，以召十方，始当诣座。
　　元犹乾元、坤元也。先乎天地而天地由之以生，是为万化之始，万汇之祖，乃虚皇天尊之应号，是为诸天之尊也。当是时，演说度人经教，周行十过，以召十方上圣高尊，大会黍珠之中，敷扬妙蕴，始乃就座而说经。夫元始即法身之祖炁也，元神也；所谓不坏真空，本来自性也。当说是经，即祖炁为丹之体，元神不昧则天命流行而不息，周流六虚，动静无端，往来不已，是乃周回十过。十过者，生数五，成数五，十乃天地之成数也。修丹以十月而后成功。十方者，乃一三五七九，阳也；二四六八十，阴也。此为还丹之妙用。而玄关之中，真阴、真阳之炁既济，则一身百脉辐奏川流，犹十方真圣不召而集矣。
　　天真大神、上圣高真、妙行真人、无鞅数众，乘空而来，飞云丹霄，绿舆琼轮，羽盖垂荫，流精玉光，五色郁勃，洞焕太空。
　　凡无鞅数众之真仙，乃五亿诸天日月星宿、天真地只、天地水府、三界十方仙真圣众，不可以数度之。一时皆乘空而来，御空蹑梵，不行而至。飞云丹霄，驾三素之云，乘绛霞之炁。轮舆羽盖，皆真圣一时之神化百态万状，灵变莫测。流精玉光，五色郁勃，流晶三光之辉，交映玉光，仙真之圆象洞明，是以五彩交烜於太空，焕明洞彻，内而修之。金母既返神室，则以神摄炁，以炁炼神，铅汞擒制，阴阳攒族，是即千真万圣之来会也。须臾，金火交辉於神室，内外朗彻，流精炫彩，五色焕烂於太空矣。
　　七日七夜，诸天日月星宿、璇玑玉衡，一时停轮。神风静默，山海藏云。天无浮翳，四炁朗清。一国地土，山川林木，缅平一等，无复高下。上皆作碧玉，无有异色。
　　七乃火数，即七返之义也。《易》曰：七日来复。凡数遇七叉复。先儒曰：阳生於子，阴生於午。自午至子，七而必复；乾坤消息之理也。以一日言之，自午时至夜半，复子时。以一年言之，自五月一阴生，至十一月复子月。以一月言之，自午日几七日，复遇子日。此天道运行之定数也。诸天，即三十二天也。日月星宿、璇玑玉衡者，周天星辰与中斗七星也。一时停轮者，天体浑沦，运转周流，无一刻之息。至是时乃停轮息转，即复归其根，中行独复也。以其静而来复，息不出入，是以神风静默，情虑顿消，气无上下，犹山川之大云雾俱藏，天之浮翳亦除，则四炁朗彻。一国地土，即浮黎土也。山川林木、缅平一等，无复高下，当说经之际，神疑目定，脉住心停，性复情冥，万机俱泯，则皆缅平一等。自性如玉之莹，了无异色矣。学者能忘形绝虑，万有皆空，体性湛然，一真独露，则一身之景象，百脉皆停，真铅现彩。是谓土皆作碧玉矣。玄师曰：此乃大定，神光湛然，见玉清之境是也。
　　众真侍座，元始尊玄坐空浮五色狮子上。
　　元始者，元神也。玄坐者，神室也。故空浮去地五丈，不着於物相也。五色者，妙化之炁也。狮子者，心知也。众真侍座左右，元始法身玄坐空浮，五炁朝於绛宫。五色朗明，即元神居於神室，冥心摄炁，三界混融，心知明妙，洞朗太 无，则五方之色交映中黄之宫矣。
　　说经一徧，诸天大圣，同时称善，是时一国男女聋病，耳皆开聪。说经二徧，盲者目明。说经三徧，喑者能言。说经四徧，跛痾积逮，皆能起行。说经五徧，久病痼疾，一时复形。说经六徧，发白返黑，齿落更生。说经七徧，老者返壮，少者皆强。说经八徧，妇人怀妊，鸟兽含胎，已生未生，皆得生成。说经九徧，地藏发泄，金玉露形。说经十徧，枯骨更生，皆起成人。是时一国是男是女，莫不倾心，皆受护度，咸得长生。
　　金丹九转之功，以十月为胎圆神化。河图生成之数，自一至十而生生无穷。洛书变用，除十纵横，一五一十，此盖天地五行不易之定数也。万物万事，岂能违哉。故说经一徧者，天一生水也。水孕於乾金，故子居母腹，是曰空炁金胎也。天地万物皆由是而生，盖水为数之始。诸天大圣同时称善者，诸天皆欣赞也。能使一炁周流，百神俱畅矣。一国男女者，犹一身阴阳之炁也。其炁既舒，聩者自聪。且坎为耳，一数既周，聋者自疏通矣。说经二徧者，地二生火也。火孕於巽木，故母隐子胎，是曰木火同位。且离为目，二数既周，离明洞焕，目无不明矣。说经三徧者，天三生木也。震木孕於艮水，且音属金，金本克木而反寄胎金乡，是木绝於申，而胎於酉，此盖木中藏金之象也。金木受制则必鸣，是以喑亦能言也。说经四徧者，地四生金也，兑金孕於坤土，故土能生金。足本属木，而四徧乃跛痾起行。金绝於寅而胎於卯，是金尝寄胎於木，此明金中有木之象也。说经五徧者，天五生土也。戊土孕炁於离火，得天地之正，流戊就已，则二土成圭，是以坎戊月精离已日光，日月皆合璧矣。人能知修炼，惩忿窒欲，存心养性，致炼己之功而不息，则太一含真，玄珠成象，与乾坤同体，长生久视，为不难矣。尚何久病痼疾之为患哉。人之困於疾厄者，由不知返还之妙也。说经六徧者，地六成之也。天一生水，坎位於北，阳中而有真阴，是以乾金生坎水而成真铅。铅乃金之母，铅黑而金白，故谓发白反黑，齿落更生；皆返本之应也。说经七徧者，天七成之也。地二生火，离位於南，阴中而有真阳，是以巽木生离火而成真汞，即七返也。离汞为金铅所制，则金火暖炼。乾金，老阳也。离火，少阴也。是乃老者返壮，少者皆强矣。说经八徧者，地八成之也。天三生木，震位於东，艮震合而成水，汞感铅炁而结为砂，砂含汞质以象怀胎，皆先天真一之炁，凝合於规中，发生万汇，昼夜循环，长养圣胎，非后天阴滓之炁所能育，故曰妇人怀妊，鸟兽含胎，已生未生，皆得生成也。说经九徧者，天九成之也。地四生金，兑位於西，兑金合於坤土，坎水隐於母胎，即九还也。故汞归土釜而成宝，乃一炁还源矣。始由空炁金胎而生，未见金质，至此则金居性初，还中宫而为丹体。中即土也。土为地，是以地藏发泄，金玉露形。金母内居中宫也。说经十徧者，地十成之也。天五生土，土位于中，土乃万物生成之本。天五之土，阳土也；地十之土，阴土也。两土重而为圭，故还丹谓之刀圭也。金居其中，是乾金反制离火於坤土之中，所谓五行颠倒衍也。人能修之，则返老还童，超凡入圣，故能起死回生，若太上之治徐甲是也。故曰枯骨更生，皆起成人。是时一国是男是女，莫不倾心。一国之中男，女等众，一闻经教，咸受护度而证长生之道矣。国，犹身也。男女即阴阳二炁也。倾心者，阴阳以类相投也。阴阳之炁，冲和则太乙归真，生身受度，死魂受炼，是谓长生矣。
