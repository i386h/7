悟真篇注释
　　经名：悟真篇注释。三卷，原题象川无名子翁渊明注，即翁葆光，出于南宋时期。底本出处：《正统道藏》洞真部玉诀类。
　　悟真篇注释序
　　夫子尝谓余曰：天台仙翁道成授命紫阳真人之号於上帝，默相皇家，时尝隐显於世，人莫之识也。翁少偕我祖肄业辟雍，唯翁不第，夙挺灵根，因玩佛书，忽生击竹之感，顿悟无生，直超真空清静性海。晚景遇青城丈人於成都，尽得金丹之奥旨，洞晓阴阳互用之机，天地反覆生成之理，故能修真复命，升入无形，抱一明心，咸臻空寂。是以形神俱妙，与道合真，变化无穷，普现法界。即兹妙用，广度群迷。逮今而有遇其性命之道者，诚为鲜矣。元丰间，与刘奉真之徒广宣佛法，以无生留偈而入寂，奉真之徒焚其蜕，获舍利千百，其大如茨。后七年，奉真之徒至王屋山，复会仙翕如故，此以示其形神俱妙也，性命两全之意。仙翕蕴性慈仁，慷慨特达，穷理尽性以至於命，三宗一致，妙用无殊，不欲独善诸身，故作《悟真篇》提诲后学。先以神仙命道诱其修炼，故以命金丹之术首咏是篇，终以真如空性遣其幻妄，故以禅宗咏其歌诗毕其卷末。金丹之要，以二八真阴、真阳之物立为炉鼎，取先天之一气归斯炉鼎之中，变成一粒大如黍米，号曰太一含真。是以首立七言四韵一十五首，以表二八真阴真阳之数。五言一首，以表太一之一粒。既得一粒饵归丹田，然后运火，依约六十四卦而行之，故吟成绝句六十四首，以按《周易》六十四卦。其运火之功有十月，并沐浴共有一十二月，又续添《西江月》一十二首，以应周天之岁纪。十月功备，胎圆而成，化为纯阳之气，故总吟成律诗八十一首。以象纯阳九九之数。形化气矣，然后抱元九载，炼气成神，以神合道，故得形神俱妙。妙而无形，神与道合冥而不测，是　命性悉居於究竟空寂之本源也，故以禅宗性道毕其卷末者也。故上皆取金丹大旨，次序如此。若乃药物火候口诀皆寓意在歌咏之中，览者可寻文而解悟。自后传之浸广，文理颇有不同，又多错谬，唯龙图陆公之孙思诚之家其本为真，此乃仙翁亲传之本也，仙翁亦自序其所得之详於卷末矣。愚因游於洞天，得斯真文，改而正诸，始悟仙翁所作之意，次序篇章取金丹之法象。其言虽约而义理该着，寓意虽微而比类亲切，诚为后学之真规，群经之要览。今夫学者多取旁门非类而证之，或以天庭至宝、玉壶金丹、混元灵丹之类妄乱穿凿，终莫证其深根固蒂而具者。又有叶文叔者，以太极大衍之数释而辨之，复撰图附於卷末，谓之《悟真篇外传》，此乃帘瞻幙影，定马为乾，非惟素乱真经，而使学者愈增疑惑。
　　噫，仙翁有言曰：不解自思己错，更将错路教人，误他永劫在迷津，似恁欺心安忍。非文叔之谓乎。焉知金丹一粒，即太极之一炁也。圣人假借二八之物，采其气於一时之中，变成一粒饵之，则立跻圣地。仙翁曰：一时辰内管丹成。又曰：一粒金丹吞入腹，始知我命不由天。岂虚言哉。文叔不达斯理，反以一时为非。若非止用一时而言之，是非三年必九载矣。为至简至易，而仙翁有都来片饷工夫，永保无穷逸乐之语哉。又以一粒而为一日之丹，妄引真一子有日食一粒之说。然则三年九载，必食千有余粒，岂仙翁特以一时之语诳人乎。举此两端，足知文叔不得金丹之术明矣。是以妄乱笺注，纰谬非一，殊不知太极大衍之数其实运火托象之时也。若托象求金丹之至道，是犹象龙致雨，画饼充饥，不亦难乎。愚固不敏，陪仙翁之遗躅，既承真应，寅夕不忘，安敢坐视红紫乱朱而不能广仙翁之辞意而明之者耶。是以不惧天谴，直泄天机，谨依师之秘旨，课解真文。分为三卷，上卷言其强兵战肚之卫，以采其金也。中卷言其富国安民之法，以运其火也。下卷言其神仙抱一之道，以入元形也。一一说其实事，罔有纤微遗漏，同志览之，坦然明白，直证至道，而不昧於邪宗曲流也。无名子谨序。

　　象川无名子翁渊明注
　　夫炼金丹，每以中秋初刻，一阳动时，坐镇魁罡，坛升三级，左擒龙而审定鼎弦，右擒虎而精调气侯。一文一武，争交战於玄门。一去一来，互斗危於牝户。息符刻漏，数应周天，无令毫发差殊，纤微悔吝，故得片饷时中，三战才终，立夺丹珠入口矣。故曰强兵战胜之术也。战者，交媾也。兵者，龙虎也。不一时辰获金丹入口者，战肚之术也。故《西华经》曰：强兵战胜究在於养玄珠之功也。是以仙翁首列七言二八首者，以明龙虎之数也。五言一首者，以表一时辰得金丹一粒也。此愚所以为上卷表而出之，则强兵战胜之义也。采金丹之功，焕然而明白矣。
　　七言四韵十五首
　　不求大道出迷涂，纵负贤才岂丈夫。百岁光阴石火烁，一生身世水泡浮。
　　为贪利禄求荣显，不顾形容暗瘁枯。试问堆金等山岳，元常买得不来无。
　　人间所重之至极者，日富与贵是也乃人之所欲，故天下之人莫不快其性命之情，尽其平生之志，争先力求之为快。观其所以然者，无过浸淫利禄声色，实为伐性命之戈矛也，为此身之桎梏也。何则，夫世人不明道德之心、性命之大，唯贪利禄，日恣真痴，汨没爱河，漂沉欲海，是非人我，交战胸中。喜怒哀乐，互残躯内。是致尸鬼促其年寿，寒暑消其容光。不觉身生一世，瞥然水上之涯。光景百年，瞬若石中之火。纵使积金齐斗，玉垒等山，逮至元常，而欲买身，使不为蝼蚁窟穴，可乎。哀哉，痛哉，命未告终，其灵已投於别壳矣，虚靖真君曰：今生不觉，换入别壳，转转不觉。呜呼，与其投身於异类，曷若栖迟於大道。若道遂功成，身超碧落，乘云炁，驭飞龙，而游乎无极，永世不变乎己，位号真人，此大丈夫得志之秋，至乐至耀之日也。若区区俗务，碌碌尘心，而坠於世网者，纵负孔孟之贤才，兼有苏、张之荣耀，不过为土上之游魂，行尸之阴鬼耳，乌足以为真丈夫哉。是以仙翁首咏是篇，盖为特达高明之士而言之，可因一言而自悟，速求大道，出离迷途，为无为之事，乃真丈夫耳，除此俱无足取矣。
　　人生虽有百年期，寿夭穷通莫预知。昨日街头犹走马，今朝棺裹已眠尸。
　　妻财抛下非君有，罪业将行难自欺。
　　人之寿夭、穷通、富贵、贫贱，未尝不默定於本然有分之数，岂可测而预知哉。夫人之寿夭约百年，近者七十固已稀矣，今以有限易摧之身，终日役役，以逐无涯，不亦劳乎。一息不来，则薾然长往，不知所归，可不为大哀耶。故有昨日方走马，今日已眠尸，出息不保入息者也。当斯之际，虽荣居极品，禄享万锺，家丰无价之宝，室美倾城之艳，皆悉抛下，为之一空，非己之有也。所有与之偕行而不可欺者，平生所造业罪而已。故云万般将不去，唯有业随身者也。曹真人词曰：叹人生，多忙乱，火宅尘绿日相牵绊。蓦地喉中三寸断，性魄神魂自此俱消散。任妻儿哀切吹，万句千声更不回头看。饶你在生多计算，卧在荒丘失了惺惺汉。诚哉是言也。
　　大药不求争得遇，
　　夫人欲免轮回而不堕世网者，莫若金丹大药为升天之灵梯，超几之捷径也。其道甚简甚易，虽愚昧小人得而行之，立跻圣位矣。奈何而上圣秘重，不许轻惇（亻＋享）漏泄，唯口诀授贤，不记文字，是以难遇而易成者。自非勤求苦志，诚动高穹，未闻有一二得者。其谢自然以兹道之难遇，思慕真师於蓬莱，是以竭其精诚，倾囊倒廪，悉备舟楫，不顾洪涛巨浸之危，直往而不少惮，遂感海神而语曰：蓬莱弱水三万里，一芥不为之浮，子将安往。赤城山有司马子微居焉，子往师之。於是回舟，寻访赤城，果遇子微，授其道，修炼不数载，白日升天。噫，精诚发之於中，感格应之於外，则无所不至也。若能操心立志之如此，奚虑金丹之道不成邪。道不负人，人常负道耳。《参同契》曰：夫道无适无莫兮，唯付与贤者。倘不坚诚力慕，争得遇之哉。
　　遇之不炼大愚痴。
　　金丹秘要诚难遇矣，得遇之者，宿有仙骨，祖宗阴德厚也。又须巨有财力，结友三人，方能就此理。在达者自知固难言也，是故王冲照遇海蟾公得金丹之道，无财下手，遂入洛谒韩富二公，赖有力者成道而去。苟有遇此道，又得有力者同心修炼，而不肯为者，实愚痴之甚也。仙翁赠刘君诗曰：闻君知药已多年，何不收心炼取铅。莫教烛被风吹灭，六道轮回难怨天。正谓是也。
　　学仙须是学天仙，
　　仙有数等，阴神至灵而无形者，鬼仙也。处世无诸疾恼而寿永者，人仙也。飞空走雾，不饥不挠，寒暑不侵，遨游海岛，长生不死者，地仙也。形神俱妙，与道合真，步日月无影，入金石无碍，变化无穷，或老或少，或隐或显，或存或亡，聚则成形，散则成气，蓍龟莫能测，鬼神莫能知者，天仙也。故阴真君曰：若能绝欲修胎息，移神脱壳，入定投尸，托阴生不坏者，为下品仙也。若授三甲符箓，太一盟威，上清三洞经法及剑衍尸解之法得道者，并为南官列仙，在诸洞府，为中品之仙也。若修金丹大药成道者，全身冲天，乃为无极上品天仙也。仙翁勉修真之士，须立慷慨特达之志，断念绝浮华，凝神乐无为，不群中下之仙，当证无上无极上品上仙也。
　　惟有金丹最的端。
　　丹有七十二品，欲学天仙，其道简而易成者，惟有金丹至道最端的矣。此盖无中生有，非天地后生五金、八石、朱砂、水银、黑铅、白锡、雌雄、砒粉、秋石、草木之类，及自身津、精、气、血、液应干有中生有等物，惟先天之前混沌真一之炁。用法追摄於一时辰之中，结成一粒大如黍米，号曰金丹，又曰真铅，曰阳丹，曰真汞，曰真一精，曰真一水，曰水火，曰太乙含真炁，人得服饵，立跻圣地，此乃无上之甲科，天仙之妙道，举世罕得闻此。
　　二物会时情性合，五行全处虎龙蟠。
