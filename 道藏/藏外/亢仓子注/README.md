　　全道篇第一【心冥虚极德洞防功并四时苍生自化】
　　亢仓防【古子字上从川发也师古曰子者人之嘉称】居羽山之颜叁【古三字或作参数名】年【年从禾千谷熟也取禾岁一也俗作年羽山尚书禹贡在徐州舜典云殛鲧于羽山盖在东裔后属鲁国颜山之南面也庄子引此章云北居防垒之山即此山是也】俗无疵疠而仍谷【圣贤所在天佑之福故俗无疵疠而五谷丰稔频曰仍】其俗窃相谓曰亢仓防之始来吾鲜【从火肰烧也借为语助作然非】异之【鲜然惊异之貌异其虚懐寂泊不存近情】今吾日计之不足岁计之有余其或圣者邪【验其利益故疑之为圣人】盍相与尸而祝之【音社】而稷之虖【虖俗作乎语助盍何不也既防厚利欲立为君主而建置宗庙并及社稷坛壝祝祭南面事之乎】亢仓防闻之色有不释其徒黡【一本黯】啜从而启之【黡啜亢仓防门人也欲知其心故从而谘启】亢仓防曰吾闻至人尸居环堵之室而百姓猖狂不知其所如【古往字之也至人防心绝虑有于尸无事萧然独居环堵苍生欣慕其往归依察其所归非由知者也】今以羽俗父子窃窃焉将爼豆予我其的【一作杓】之人邪【窃窃私议之谓也我本栖全道任真今乃爼豆相尊反成人之标的】吾是以不释【舍也】于老聃之言【老聃言我旡为而民自化我无事而民自足我好静而民自正今乃反此故不释其言】黡啜曰不者【不者犹不然也】夫寻常之污巨无所还其体而鲵防□【音为】之制步仞之丘巨兽无所其躯而防狐□之祥【八尺曰寻倍寻曰常污池也还也鲵小鱼而有脚制犹专擅也六尺曰步七尺曰仞防妖也详善也言小水不能容巨鱼小丘不能藏大兽喻亢仓子道德既高必须厚禄也】且也尊贤叓【音事古作叓从史之省声】能向善就利自尧防以固而况羽俗虖先生其矣亢仓防曰嘻来【嘻叹声也怪其不达已志故发嘻叹将欲告之致远故呼之来】夫二子者知虖【二子尧防也知乎言岂知之】函车之兽介而离山罔罟制之吞舟之荡而失水蝼蚁苦之故鸟兽居欲其高鳖居欲其湥【古深字从水突声浅湥之对】夫全其形生之人藏其身也亦不厌湥而已【函盈也介孤也远也】吾语若【若汝也】大乱之本虖尧防之间其□【音终】存虖【音乎】千代之后必有人与相食者矣【事有先成后败始吉终凶胡可必邪故尧舜禅让而光一时之美迹流后代成篡弑之祸故庄子云尧舜让而帝哙之让而绝汤武争而王白公争而灭斯其效欤夫惟不立善名者则事迹冥绝无所企慕耳】言未终南子荣之樗色蹴膝席曰樗年运而长矣将奚以托业以叓【音事】斯言【荣之樗庄子所谓南禜趎也既闻高谊将欲蹴然变色敛膝于席愿垂告示以敬事此言】亢仓防曰全汝形抱汝生无【古悖字从子乱也逆也】汝思虑营营若此绪年或可以及此言【营营运动不息也绪终也全形抱生不运思虑虚心冥寂道自居之比此终年可及此言也】虽吾才小不足以化防防胡不南谒吾师聃【聃老子名】亢仓防既谢荣之樗不释羽俗而龙化虖天下【谢遣也不释羽俗潜遁而游如龙变化与时升降】水之性清土者抇【胡骨反】之故不得清人之性寿物者相之故不得寿【抇乱也人性寿考为外物所乱使不终天年】物也者所以养性也今代之惑者多以性养物不知轻重也【衣食养性不可一日而无而惑者乃损性以求物物愈积而性愈伤殊不知性重物轻倒置也】是故圣人之于声色滋味也利于性取之【音害利害之对】于性捐之此全性之道也防【古万字数名】人操弓□【音共】射一招【又音雕】招无不中防物章章以一生生无不伤【捐弃也操持也招射的也章章犹扰扰】故圣人之制万物也全其天也【圣人抑制万物不使伤性以全天真】天全神全矣神全之人不虑而通不谋而当精照无外志凝宇宙德若地【古地字从土彖声与埅同】上□防而不骄下□匹夫而不愍此之谓全道之人【神全之人智虑充溢志气凝乎宇宙覆载之德同乎天地虽贵为天子贱为匹夫不以穷达而回其志】心平正不□外物所诱曰清清而能久【古则字从鼎刀作则】明明而能久虚虚道全而居之秦佚死亢仓防哭之【秦佚古之有道者盖老子之友也】其役曰天下皆死先生何哭□也【役门人充使役也死生之道古今是常达人休之不哭可也】亢仓防曰天下皆哭安得不哭【顺物而哭虽哭而非哭】其役曰哭者必哀而先生未始哀何也【未始未尝也世人之哭必生哀痛今先生虽哭不见哀伤敢问何故】亢仓防曰举天之下吾无与乐安所取哀【夫有乐必有哀人之常情也达人大观岂有疎亲既不与为乐亦无所取哀】蜕地之谓水蜕水之谓气蜕气之谓虚蜕虚之谓道【蜕者免蜕之谓也夫蜕地之谓水蜕水之谓气蜕气之谓虚蜕虚之谓道犹至人不系情于哀乐然后为极者也】虚者道之体靖者道之埅【音地】理者道之纲识者道之目【言至人能虚能静有理有识者则能契道之形体知道之纲目】道所以保神德【古德字从心直声有德者心直】所以宏量礼所以亝【音齐平也】物物所以养体【四者皆可以资身不可暂无】好质白之物者以黑□污好质黑之物者以白□污吾又安知天下之正洁污哉由是不主物之防污者矣夫瞀【莫豆反】视者以黈【偷厚反】□赤以苍□吾乃今所谓皂白安知识者不以□赪黄吾又安知天下之正色哉由是不遁物之色矣【瞀风也黈黄色也赪赤色也夫有风之疾者视物不能审也故以黄为赤以青为黑亦犹凡俗之情妄执防污虽有大圣亦孰能正之故不生其洁污不流遁于众色】夫好货甚者不见他物之可好好马甚者不见他物之可好好书甚者不见他物之可好吾又安知天下之果可好者果可恶者哉由是不见物之可以保恋矣无能滑吾真【音真】矣陈懐君柳使其大夫祷行聘于鲁【陈懐君谥柳名也祷大夫之名】叔孙卿私曰【叔孙其氏世为鲁卿】吾国有圣人若知之乎【圣人谓仲尼】陈大夫曰奚以果明其圣叔孙卿曰能废心而用形【圣境超殊非凡情所测徒见其应接世务便证以为圣人岂知其所以圣哉】陈大夫曰弊邑小亦有圣人异于所闻曰圣人谓谁陈大夫曰有亢仓防者偏得老聃之道【门人中最为称首故曰偏得】其能用耳视【古视字从目示声】目【古听字从耳壬声】定公闻而异焉使叔孙氏报聘且【音致】亢仓防待以上卿之礼亢仓防至宾于亚寝【亚寝公之次殿】鲁公卑辞以问之亢仓防曰吾能视不用耳目非能易耳目之所用告者过也公曰【音孰本作□】知是【一本孰如是】寡人增异矣其道若何寡人果愿防之亢仓防曰我体合于心心合于气气合于神神合于无其有介之有唯之音虽远际八荒之表迩在眉【古眉字从目上象额理】睫之内来于我者吾必防知之乃不知□是我七窍手足之所觉六腑五脏心虑之所知其自知而已矣【心形泯合神气冥符洞然至忘与符同体然后心亦静而志弥远神愈默而昭愈彰理极而自通不思而览非至圣至神其孰能与此斯固灵真之要极重之妙道也】
　　用道篇第二【无非利物上合天心克已归仁化行刑措】
　　不可信地【音地埅同】不可信人不可信心不可信惟道可信贤主秀士岂知哉昔者桀信与其四海已不勤于道夺其国以授殷纣亦信与其四海已不龚于道夺其国以授周【与犹以也祖主也夏桀殷纣体淫奢纵自谓有民有命禀在上天穷凶肆虐不修其道故天夺其国以授于汤武】今夫惰防【音农从防防声耕也】信地实生百谷不力于其道地【亦地字地埅同】窃其果稼而荒翳之【惰农之人信地生谷不勤耕耘稂莠荒芜故不能获果种】齐后信人之性酬让不明于其道举全境以付人人实鸱义而有其国【齐简公信人性不明酬让为陈恒所弑人取其国鸱义贪残也】凡人不修其道随其心而师之营欲茂滋灾疾朋衅戕身损寿心斯之矣【人自师心不遵圣教营欲炽盛百殃斯集此乃心为身害朋群也衅动也戕害也】故曰惟道可信【道者坦荡怙怡无所染着人能虚心归道则身命安全】埅非道不能悠久【天得一以清地得一以宁】苍生非贤不能靖顺庶政非才不能龢【古和字】理【贤才皆用道以理物】夫用道之人不露其用畗【古福字从田高省有之义】滋防【古万字】物功归无有【潜功密济百姓谓我自然】神融业茂灵【古灵字善也】庆悠长【融通也神理通达德业荣茂厚善所积庆流后裔】知而辨之谓之识知而不辨谓之道识以理人道以安人【辨析事物使人去恶就善所以理人也含冲寂无所毁誉所以安镇浮竞也】夫鸡曟【音辰从晶辰声】而作负日任劳流汗洒埅夜分仅息防夫之道也【鸡辰鸡鸣之旦也夜分中夜也负日谓日所曝也仅少也】俯拾仰取锐意锥撮力思抟精希求利润贾竖之道也咽气谷神宰思捐虑超遥轻举日精链防高士之道也【燕气胎息五牙之也谷养也宰割也日精吸日精也链仙链质而升仙也】剸情端【作端非正也】想毕志所吏伦揆忘寝谋效位司人臣之道也【专情正想尽志于所事之君遵理揆度效功于所司之位是人臣之道也】清心省念察验近习务求贤良以安防姓人王之道也【清心寡嗜欲也省念无私也近习近臣也】若由是之各顺防【音序恐是防字误】其志度不替□【音塞】其业履是谓下有道【农夫贾竖各保其业明君贤臣各修其道则天下顺序而业履安定矣】导筋骨形全剪情欲神全靖言语畗【古福字】全□【古克字能也】保三全是谓清贤道德顺神助信义□君子全礼义备小人懐有识者自是无识者亦自是有道者静默暗钝者亦静默物固有佀是而非佀非而是先号后笑始吉终凶身可亲而才不可亲才可敬而身不堪敬敬甚不亲亲甚不敬亲之而疎疎之而亲恩甚怨生防【古爱字从心旡声恩也】多憎至有以□【音为】防有以缓□防有以直□防有以曲□防百叓之其由甚□不可不知是故智者难之静神通穷意通防语通富身通理势使同道者相防同艺者相嫉人情自也【情通无求则相爱争能好胜则相嫉势使然也】多才而好谦贫贱而不谄处劳而不□辱富防而恭勤可谓有德者也
