　　儒门崇理折衷堪舆完孝录
　　经名：儒门崇理折衷堪与完孝录。撰人不详。八卷。底本出处：《万历续道藏》封字号。
　　完孝录凡例
　　一堪舆论峦头者，《雪心赋》、《葬经》等书，言之备矣。论星气者，《催官篇》、《玉尺经》等书，言之备矣。更复何赘？是编中所载，皆因古人所未言者而言之。或古人言而未详者，而详言之。一切陈言不载、但其中如广□熊氏、饶安李氏、括苍季氏、泰和周氏、临川吴氏之类，言皆切理，尚未广传于世，故并录之。
　　一论形气，诸前贤格言，皆可备参考。至七政造命年月，世所罕传。故编中论形气者略，而论七政者详。论形气者止存今文，论七政者并存古文。
　　一天符紫白，阐五运六气之机，发洛书河图之秘。其理与七政相表裹者，此亦世所罕传，故并存之。
　　一仁人孝子之心，原无所为而为。而编中辄言吉凶祸福，后来之应验者，不亦惑世乎？然以纯孝之士，不怛有於世，故举其验，以故动之，不必一一如所云。
　　一地理本静也，古人制作，每於静中求动，故多用变。夫变者，权也。权非圣人不能用，时师而误用之。其不败，乃事几希。是编惟道其常而已。至用变而涉奇怪幽玄，皆所不载。

　　儒门崇理折衷堪舆完孝录卷之一
　　第一章论阴阳正宗
　　盖自有天有地，分阴分阳，至理流通於两间，元气运行於万古。昏昏默默，妙阖辟於无穷，混混沌沌，神变通於莫测；或敷而为二气之推迁，或布而为五行之生克，一消一长，总妙玄工。为盈为虚，无非造化。天地之神固不可知，推迁之数则有可考。为日月，为星辰，其灿然於上者，有一定之度。为山川，为土石，其凝然於下者，有一定之形。推其度，可以知年月日时之吉凶；察其形，可以庾龙穴砂水之祸福。道参三极，理无二途，惟三才合一之士，可以凝天命佑神工。彼拘挛执一之徒，安得肩鸿巨当重任，高其谈阔其步，依然远类乎杨曾，拟其象画其图，居然自安於廖赖。若问天文地理，那有实见真知。大者勿论，乃拘拘於小节之疵，全体罔闻，顾屑屑於一偏之见。已昧乎？山川向背之分，且以方向言祸福，既疏於圣正授时之令，更以杂说论日时。况伪书之纷纷，济诞说之喋喋，天无二日，古今定理。今之论太阳者，则曰历数，曰乌兔，曰都篡之类，有七位太阳之参差，民无二王，圣贤格言。今之论帝星者，则曰尊帝，曰天皇，曰口斗之类，有十三帝星之间杂。变乱阴阳者，则有为争阴、为争阳、为分阴、为分阳、为真阴、为真阳、为先天、为后天，讵知太极之阴阳无二理。易置五行者，则有为阴府、为洪范、为八卦、为双山、为玄空、为纳音、为浑天、为星宿。谁知生成之五行无两端，似此之类难以悉举，信此之辈举世皆然，是以疑信交驰，令人一二世而无葬地可否难辨，致人数十年而无葬期。始信伪术不足凭，惟有吾心为真，主吾心果善，人自杰而地自灵，念虑少亏，地不庇而神不佑。不然，胡龙头政严灾兆十日，胡牛眠山上，位至三公，配兴衰成败之机，莫非天定。故贪谋诈求之计，枉自心劳，或侵掘己祖，犹如剜肉以充饥。或偷葬人茔，无异盗铃而掩耳。奉劝世人，无如守正培心地，乃为得地之本。循天理，诚为地理之根，忠不信明师，毋惑幻术。相地以五星峦头为先，而卦例天星奚足，惑择期以七政要旨为重，而百家杂说何足凭守，此阴阳正理，可参造化玄机。相地以五星四句，是一部大旨。
