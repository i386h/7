古楼观紫云衍庆集
　　经名：古楼观紫云衍庆集。元朱象先编撰。三卷。底本出处：《正统道藏》洞神部记传类。
　　古楼观紫云衍庆集卷上
　　句曲朱象先集
　　大唐宗圣观记
　　夫至理虚寂，道非常道，妙门凝邈，无名可名。爰自太始开图，混元立极，三才奠处，万品流形，莫知象帝之家，未赌谷神之域，希夷琐闭，淇滓封奇，及夫乌迩勃兴，隐书诠奥，至化因兹而吹万，玄教由是以开先。圣圣袭明，道德授受，于是混元之教，风动天下，水行地中矣。宗圣观者，本名楼观，周康王大夫文始先生尹君之故宅也。以结草为楼，因即为号。先生禀自然之德，应玄运而生，体性抱神，韬光隐耀，观星候气，物色真人。会遇仙轿，北面请道，二经既演，八表向化，大教之兴，盖起於此矣。兹观中分秦甸，面距终南，束眺骊峰，接晴岚之浥浥，西顾太白，集积雪之皑皑。授经之古殿密清，路牛之灵未特立，市朝屡易，仙迩长存，物老地灵，每彰休应。卿云日覆，寿鹤时来，树无窠宿之禽，野有护持之兽。文始药井，韩梵未堕，老君辈车，确然不朽。至於穿寄盗窃，进退自拘，似有挚维，悉皆面缚。昔周穆西巡，秦文束猎，并枉驽回辕，亲承教道。始皇建庙於楼南，汉武立官於观北，崇台虚朗，招徕云水之仙，闲馆错落，宾友松乔之倡。秦汉庙户相继不绝，晋宋谒版，于今尚存，实神明之奥区，列真之会府。后魏文帝变夷风於华俗，立仁义之纪纲，崇信教门，增置徒倡。有陈先生宝炽，颖川人，夙有幽逸之姿，幼怀林壑之趣，松风入赏，名岳留连。玉皇之道，既弘银榜之宫云构。续有王先生子玄，言穷名象，思洞隐微，念在玄空，累非外物，含神自静，仪圣作师，并德音孔昭，郁为宗范。周太祖定业关内，躬受五符；隋文皇沐芳礼谒，获闻休征。迨隋德将季，政教陵迟，六飞失驭，四维圯绝，夷羊在牧，辈巷满野，家习兵凶，民坠涂炭。皇帝命世应期，荣镜区宇，戡难静乱，亭毒无垠，广大配乎天地，光华方诸日月，数阶庭之莫荚，聆凤和呜，照景星於玄云，观麟郊薮，缉礼裁乐，化俗移风。农夫劝於时雨，陇余滞穗；工女勤於蚕绩，杼轴不空。九服韬戈，三边静拆，西戎革面，束夷献舞，朔南洎声教，漠北尽来王，德化遐渐，无幽不畅，三善克懋，非假二疏。一有元良，万邦贞固，昭一均天纵，道契生知，笃尚玄根，钦兹圣躅，以武德三年诏锡嘉名，改楼观为宗圣观。宸宸兴念，纂冑所先，启族承家，鼻於柱史，得一以灵，蹈五称圣，弱为道用，柔为至坚，损之又损，以至于益，瓜艇绵长，庆源悠侵，爰初启祚，致醮灵坛，自然香气，若雾霏空，五色云浮，如张羽盖。七年，岁惟作噩，月在黄锺，六辔齐酿，百辟咸从，亲幸观所，谒拜尊仪。轩后之诣崆峒，神农之上石室，顺法行礼，异代同规。观主岐平定，精金格之书，究玉岌之文，知来藏往，尽化穷神，豫鉴天休，赞弘景福。法师吕道济、监斋赵道隆，玉器凝润，鹤情超辽，辨析连环，辞同炙辗，对歇天旨，妙沃帝心，乃谓片言小善，尚题钳碣，蚓夫皇舆迂驾，抱酌希微，大道资始，炉锤万物，不有刊勒，其可已乎。侍中江国公陈叔达，朝宗羽仪，词才冠秀，奋兹洪笔，为制嘉铭，其词曰：眇矣灵化，玄哉妙门。飞形九府，炼气三元，黄庭秘录，金格微言。玉京留记，金鳌还魂，扬尘束海，问道西昆。物色函关，存容清庙。建标伊始，层坛云峭。绮井虹伸，风窗电笑。玄都正律，帝台仙召。抱髓扪星，餐霞引照。豁虚罔象，无名至要。高厢久县，清泉余疗。宅心胜倡，游息众妙。绝壁翠微，漂流丹窍。鞠草如结，周原甚突。圣道将弘，重光显曜。明明我后，积德累功。陶涎寓县，叱吃雷风。庸稽大室，礼盛邓宫。时乘正位，道配玄穹。四维载仰，百世斯隆。有截示外，无思自束。祥符浃远，瑞采澄空。百神咸秩，千龄是崇。宗玄壮观，诏跸康庄。云行辇道，吹发山梁。飞文协一，接礼神皇。五旌回首，六辔齐酿。宸仪展敬，享福无疆。巍然高碣，播此遗芳。给事中骑都尉欧阳询撰，武德八年二月十五日建。
　　大唐宗圣观主银青光禄大夫天水尹尊师碑
　　闻夫真人者出巨壳，历倚杵，骑辈廉，从敦围，臣雷公，妾密妃。朝擢发於汤泉，夕晞首於旸谷，仍丹丘以长啸，戴翠华以高游一自非殖因旷劫，肃恭大浩从事於金房之前，镂心於玉晨之上，携青童而应黄录者，奚以成后来之妙相乎？继绝景而胤希声，则尊师其人矣。尊师讳文操，字景先，陇西天水人也。后秦尚书仆射纬之后。纬仕长安，故为扈人焉。若乃郁为帝师，降迩於唐勋之代，光乎王佐，应命於周武之朝，家籍代资，可略言矣。曾祖洪，宇文朝商州长史。大父舒，隋文州别驾。昭考珍，皇朝散大夫，以先知授。尊师特禀异气，垂实冥华，始降迩也，其母袁氏夜梦玄妙玉女授九老丈人之符，寤而记之，每存思也。数月而闻腹中诵经声，且时时有异光绕身矣。及载弄之始，目光炯然，眸子转盼，若有所见矣。袁氏以其所梦，有征心志而不言也。及胜衣之日，自识文字，惟诵《老子》及《孝经》。乃曰：此两经者，天地之心也。此后见好杀之字，若蹈水火，视无礼之文，如坠泉谷。稍长，闻有尹真庙，乃精心事之，不近俗事。因读《西升》、《灵宝》等经，渐达真教，既得玄味，便契黄中。闻师者传道之父母，行道之神明，无数劫来，妙经是出，不因师学谓之长昏，遂章惶无己，求师不暇。时有周法者，内音之先呜，上皇之高足，乃愿参轩效驾，陪景啸空，奔走礼谒，以申宿志。周法见之，乃谓尊师曰：汝於劫会之中，已受龟山之录也。便训以紫云之妙旨，授以青羽之隐法，一入其心，谓赤松、王子乔可与抚烟月矣。年十五，道行已周，有名于远近矣。属文德皇后遵上景而委中宫，于时搜访道林，博采真边。尊师即应玄景，行预绿云，奉敕出家，配住宗圣观。虽剪芝琼园，采琳玄陇，意每远出，未近谢也。将欲沐浴束井，栖迟南昌，保护昆仑，窥奇浑沌矣。故属想丹烟，游心紫度，褊寻五岳，备涉九元，寻三君之祖气，成七晨之慧眼。旋谒周法，便居终南。寂虑於温泉，冥精於寒谷，有年日矣。既通八景，又达九天，知来藏往，多所晓悟，若有神日：周法上迁。及省所居，已去顺也。贞观末年，行丧既毕，永徽三年，乃游太白，入重玄也。见所未见，闻所未闻，此后丹字紫书，三五顺行之法；扶晨接昼，九六逆取之方，咸得其要。尊师所有游山异边，祈醮灵应，并有别录，此不载之。至於显庆以来，国家所赖，出入供奉，询德谘量，救世度人，转经行道，玄坛黄屋，帝座天言，束都西京，少阳太一，九城二华，展敬推诚，三十余年，以日击月，始终不绝，有感必通，凡是效验，君臣同悉，救书往复。日月更回，神道昭彰，岁时交积者，不可具载，并传於帝居，一二要者，略举其目。初，尊师游，太白高顶，云雾四周，声振万壑，此处缺六字#1千仞，复有像充九色，其高十仞，欣然长往者，意已笃焉。高宗之在九成宫，有孛彗经天，长数丈，以问尊师，尊师对日：此天诫子也。子能敬父，君能顺天，纳谏征贤，斥邪远佞，罢役休征，责躬励行，以合天心，当不日而灭。上依而行之，应时消矣。是故高宗以晋府旧宅为太宗造昊天观，以尊师为观主，兼知本观事。仪凤四年，上在束都，先请尊师於老君庙修功德，及上亲谒，百官咸从。上及皇后、诸王、公主等，同见老君乘白马，左右神物，莫得名言，腾空而来，降于坛所，内外号叫，舞跃再拜，亲承圣音，得非尊师之诚感也。由是奉敕修《玄元皇帝圣纪》一部，凡十卷，总百十篇，篇别有赞。时半千为尊师作也，纪赞异秩，缮写进之，高宗大悦，终日观省，不离于玉案，乃授尊师银青光禄大夫，行太常少卿。尊师固让，不得已，辞官而受散职焉。永淳二年，天中有望，告成有日，万乘雷动，千骑风驰，天子乘阁道而御帝车，草官陪六仪而承七曜，将礼于天枢，幸中岳也。金绳未举，玉检犹潜，而六龙顿辔，三光敛色，圣体不安，旋于皇极。属紫微虚位，白云上征，万国号诉，四方遏密，太后谘访尊师，尊师日：真坊仙境，亦着代谢，物有荣悴，气有初终，大道之常，幸康神器。陛下宜存思谅间，极想钦明，密理百神，潜花万姓。文操人问地上，物裹天中，所有灵明，倍百祈请，亦望二十四结，火烧而忧尽，七十二教，水炼而法成，皆见先征，以明后事。乃着《松惑论》四卷，《消魔论》三十卷，《先师传》一卷。垂拱四年，将宾玉帝也，上足时道成咸愿奏章以延福荫，尊师止之日：有顺宜遵，不可犯禁。言讫委化，颜色如常。粤以长寿四年四月十四日，迁兆於终南文仙谷。弟子侯少微等，追思龙汉远慕龟岩，冀德音与天地同久，神道共阴阳齐化，昭花骞林，冥滋柏树，俾斯贞石，文若三光。其词日：去矣。大仙，悠哉上玄。玉谷白芝之座，金．阙紫兰之前。既啸景於琼札，固交欢於碧泉。出三万六千之厚地，入三万六千之远天。咀九华之翠菊，坐五色之红莲，常吟外景，每握内篇。春霞飞乎绛云，秋风生乎紫烟。裴回高黄岭，顾步太白巅。三秦四塞帝王国，京兆长安龙凤川。煌煌兮四明路，浩浩兮八景年。今已向上袭前果，何时来下降宿绿。当乘道之气，应传道之味。必使氤氲六合中，自然昌扬，万劫通。稽首空歌步，愿得乘九素。天下同此心，非独骞之林。银青光禄大夫、行太子右谕德兼崇文馆学士、上柱国、平冻县开国公员半千撰。开元五年十月二日，弟子侯少微建。
