　　救急解计章
　　凡人有年灾月厄，算尽相临、本命禄绝、行年妨害，宜奏此救急口章，灾难消除，福禄立降。昼东向，　夜北向。
　　银镇一双，算筹一百二十枚，青丝缕一丈二尺，信钱一千二百文，命米一石二斗，命缯二丈四尺。
　　次发炉
　　次长跪出官
　　泰玄都正一平炁，系天师阳平等二十四治，臣某稽首顿首，再礼
　　上言：谨案文书，今有某州某县某乡某里某观弟子某，行年若干岁，户口若干人，居住此处，即日叩头，请福乞恩词，肉人素质胎生，百官子孙，千载幸会，得奉
　　太上大道诸君丈人，受命皇极，自保生治，良由肉人某阖门大小顽愚，施行多违，招延不利，比年已来，轗轲不安，某家口衰厄，钱财耗散，妖怪竟起，恐有某三灾九厄，顿集躯形，为天罗所缠，地网所绕、寿命短促、年等穷讫、死名解奏三官，又冢墓土绝，令先亡扰动为殃，注复连所见，讼逮生人，且今世浊恶，百邪凶暴，鬼贼交行，百病杀害，平人死者狼藉，哭声盈路，不可听闻，肉人处在其中、无功自拔，深恐奄充灰壤，不自存保，无方为计，推以一心归命
　　大道告诉、向臣求乞章奏、解除某生命所犯三灾九厄、罗网缠缚，请赎应死之命，如所列诉，臣以下愚，不明鬼炁、恐某身为六害，更相克贼、又恐妖□所见侵枉，又为先亡后死，中外鬼贼冢墓之炁所可列逮，太山府君所可横召、某诚复罪秽，但雀鼠之生，从道求哀，伏闻太上大道，慈育群生，无不救拔，延命度厄，惟在恩覆置词请命，露出中庭於太阴之下，北斗之前，伏地拜奏口章一通上闻
　　天曹，伏愿
　　无极太上老君、太上丈人、三师君慈父
　　圣母，丐无极之恩，原除某年生已来所犯万死之罪，万愆之过，九厄之期，一切除荡，乞某一生一活，一度一脱若三命已尽筹纪讫穷，特赐其短中得长、败中得成，危中得安，厄中得济，回停漏刻，申借一期，请某本命生月神部，各领千乘万骑，救解延命之厄，即为排天，乃致书乞丐子午之算，倍年百二十岁，乞为肉人代名易形，削除恶簿，易短为长，愿转化推移，展转穷年，推於魁纲之下，更相拔赎，若某年命薄弱，当为延续，仓谷乏少，增其斗斛，树木枯槁、为传膏泽，若三寿短促，增其延续，若年命不长，增其算禄，辄劳神兵功曹使者，责持某年命簿帐目录，上达
　　玉帝，几前随事，为对司命门下，分别料度简书、乞除刑止杀、若某身寿有限，其度促短，九厄并会，身行多违，脏满罪定，应允驱除，三命欲尽，三解已报，玉历无籍，左契无名，三元无录，司命辍籍，南宫不收，天帝不原者，乞丐除之，三月一时，三岁一期，都会周而复始，如蒙省理，则受师老再生之恩主，为某上请文昌宫中寿命君，次请南昌益算君拔命君，各五人，官将各千二百人，一合来下，主为某延请生命，增益寿算，永保存活，又请
　　太上玄元始黑官中请条星度算君、转名易形君、脱死上生君，各五人，官将千二百人，一合来下，与臣某身中功曹使者，太阴之下考召君吏，为肉人某，上诣
　　北斗御女宫中主为拔赎死籍，转名易形，又请东华宫九龙符命告下五岳仙公，勑水官河伯吕公子等，太山二十四狱缘吏。三台，八座司命，解除某身中罪，考一切和释，为拔出三魂七魄付与身中，不得放逸，请拘魂制魄君五人，官将千二百人，又请东方七童子，拔魂魄吏万二千人，为径诣玄阴御女宫中，主为拔赎某三魂七魄，拔出三命，永保生活，又请南昌主真君五人，官将千二百人，一合来下，随臣章上诣
　　太浩天曹司命韩君门下玉案上，主为某却三官死解，更定生名，度着左契玉历神仙簿中，延年无穷，又请魁纲君五人，官将百二十人，主为某身上解生月胞胎，生日所犯五星二十八宿，下解孤虚灭没之厄，五灾九厄之炁，赐纯阳生炁布润某身。所苦千灾万痛，应章除差，恐立命以来犯触天地四时。五行十二时辰，二十四神，天纲河魁，太一从魁功曹，传送大吉小吉，胜先神后太冲，征明青龙白虎，朱雀玄武，螣蛇天后，太阴天一，贵神日游月行，千禁万忌，殃考深重，应充鬼伴者，愿七星君，次为消除之，贪狼巨门，除断鬼源，禄存廉贞，代名易形，文曲武曲，除却死录，天纲破军，斤却妖群，使不加人，三台主除墓，考千罪万过，十二刑杀之鬼，一切消灭，所患除愈，愿上官典者，为通行省理，令章上诣
　　三天曹，分别某家昔世已来，九玄七祖，父母及某身积行所犯顽逆丑恶，非人所行，淫色盗窃、违天负地，形梼咒诅、复连之炁，并肉人某身行年本命诸灾厄衰祸之炁，次解除五刑六害，短寿之厄，延年益等，六甲六十日旬，阴阳六十神，各自驱使，捡统四海八极，十四州九十郡，一千二百县、万二千乡亭、万二千里域之中，推某身所犯，并是何罪何事，在何宫在何司，何府何神所喷，为何鬼所言，为何考所逮，乞所召君将吏兵，上诣
　　三天曹道行之处，次为和释消灭之，使天无文书、地无简籍、水无盟誓，一切为原之。重请仓火玉女千二百人，各衣珠玉杂色之衣，三环结角，主为某延年益寿，添仓益库，更加寿等，起百二十神炁，流布某身中，上升泥丸，下治绛宫，养育根元，补护年命，乞医理疾，病豁然除差，登行道路，又请玄老大将军十万人，玄老使者百万众，又请三天太上玉女千二百人，三环结角身着五色玄黄衣，足着师子履，三天上来下，口含雄黄之药，手捧玉浆，吐哺真炁，共来济疗肉人，某身从头至足，绿肌缠脉，和调六腑，筋骨充满，百病除差，梦寤真正，得度九厄之中，主为某身中上解除天厄。下解除地厄，次解三天星辰之厄，次解胎月之厄，吹解生日之厄，次解年衰算尽之厄，次解嫁娶移徙葬埋犯触十二时神之厄，次解大将军太岁太阴十二月建王之厄，次解行年绝气之厄，次解违天负地之厄，次解元辰衰否之厄，又恐某身本命众厄兼会，有五方，驱除夷蛮戎狄及中外强畔，四面咒诅，克害，致令肉人疾病厄困万痛加某身，命在时克恐不兔三灾九厄之中，雀鼠之命，畏死贪生，重请三天解厄君五人，官将千二百人，又请太皇万福君五人，官将千二百人，主为某身於九天之下、九地之上，检寻某算籍，主知祸祟根元，灾厄深浅，一切悉为和合，次消灭之解而度之，乞丐肉人某身百病去体，十善来亲，千灾消灭，万福唯新，重请南昌君五人，官将百二十人，摄录肉人身中三尸九虫，无令越逸，伺人罪过，佐助未周，牵引外邪，阴改贼病，一时消灭，愿上官与者随事料理，所请天官，以时差遣君将吏兵，各摄所部，尽除恶害，收捕邪炁，解除和释，众精魍魉，奇鬼妖魅，托形诸杂，神鬼盘固不去者，请天中敢健吏兵士千二百人，一合来下，到肉人居宅里中，收捕精祟万邪为害，令内外清洁，神明防卫，特乞某身康体健，真炁降流补润四支，宁安五藏，调理百关，厄中得度，死中得生，彻灭短籍，着名长生玉历之中，千罪万过一切赦除，蒙恩之日，臣谨仰宣妙法，劝喻肉人，修善改恶，伏从科戒，不使复犯天官，营护有劳，随臣都章，言功举迁，以为效信，恩惟
　　太上省理，分别求哀，臣愚谨因二官直使，正一功曹，左右官使者，阴阳神决吏，科车赤符吏，刚风骑置史，驿马上章吏官各二人出操，臣谨因拜救急口章一通，上诣

　　南岳魏夫人生算度厄章
　　泰玄都正一平炁，系天师阳平等二十四治炁三洞道士臣某，稽首顿首，再拜上言，谨案文书，今有某州某县某乡某里某官弟子某，即日叩头乞恩，禳灾解厄救病，断绝外□，复连安神定魄章词，但弟子某千生有幸，得奉
　　大道太上诸君丈人道法，因玄都香火以自保护，得在清化，诚用忻悦罔极之恩，但由弟子某胎生顽愚，不闲科典，奉道多违，招延考逮，比来轗轲，宅舍不安，鬼毒临守，人神虚耗，钱财支散，田蚕不收，六畜损伤，人口疾患，灾厄相仍，常常梦寐，与先亡混杂，卜决云是太祖父母、伯叔兄弟、姑姨舅氏姊妹诸房，腥臭死鬼，遽相牵引，共来为祟，致令弟子某身及家口大小，卜梦凶丑，所见非真，且见今时鬼贼流行，枉暴良善，遭罹病珍，死亡者众，弟子某家口大小，处在其间，虑不免脱，惊惶恐怖，告请无方，唯以一心，上凭
