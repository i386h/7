　　太上洞玄灵宝业报因缘经卷之一
　　开度品第一
　　大慈大悲普济真人，以业报因缘诸问太上。尔时太上道君在福德天中香积世界，坐光明七宝台上，大会说法。
　　时普济真人上白天尊曰：普济侍坐无量劫来，未尝见慈尊欢喜得如今日，施大法云，注大法雨，普令见在各得解悟，随分利益不可思议。愿藉神通，欲观无极世界善恶因缘，开示当来，令知宿命、过去未来三世之事。说是语已，礼拜伏地，仰听慈颜。
　　天尊语普济曰：汝今能发如是愿，起大善心，利益一切不可思议。感此因缘，即施九色光明，彻照诸天百亿世界、乃至下方无量国土，威神变见，洞彻无阂，皎在目前犹如指掌。
　　普济及诸大众乘此威光，得见诸天诸地善恶因缘，内外备尽，诸天之上，众圣游戏。次见诸天之下，乃至诸地受苦众生，一一分别以示将来。
　　诸天之上，见有大圣高真，项负圆明，身相具足，变见自在，大小随宜：或驾景腾空，乘龙驭鹤；或飞云步虚，鸾舆凤盖。无量无边见有飞仙天仙、神仙灵人，逍遥无为，光明备身，神通无阂。见有天男天女、玉童玉女，无鞅数众。见有凤凰孔雀、狮子辟邪，侍卫左右。见有金堂玉殿，宝台珠阙。见有天乐天华、天果天树，万种杂类，不可得名。见有毒龙猛兽，匝绕城阙。见有宝池金苑，仙人游戏。见有天厨天供，百味罗列。见有诸仙，咨问法相。见有众圣，广说一乘。见有天王，以问凶吉。见有百座千座、乃至万亿宝座，讲说大乘。见有飞仙、神仙、天仙，诸始学真，问法论难。见有旛华幢盖，百宝装严。见有土地，皆是七宝金银珠玉。见有园苑、竹林、果木、鸟兽、水池，皆悉自然。见有宫殿、音声、衣服、茵席、毡褥，并非人造。见有诸仙行道念诵，礼拜赞唱，昼夜不绝。见有飞台行馆，空裹转移。见有天衣云盖，虚内相迎。见有殿堂楼阁，忽然化出。见有饮食滋味，芬芳流溢。诸天之上，见有如此福德，快乐自在之处。复见有如此殿堂、华果，种种供养不可名目。
　　诸天之下，复见有诸天大圣乘云驾虚，持旛建节，按行众生罪目籍录。见有诸天仙圣分身变化，或贵或贱，或贫或富，或病或壮，或男或女，或老或少，或大或小，或长或短，种种托形下在人间，教化一切。见有诸天仙人灵人，及诸飞天神王、善恶童子，日日游行人间，条录善恶以奏诸天。见有云宫星府、日月宫殿、山川岳渎灵官，考较人民善恶功过。
　　见有国王舍身布施，以求真道。见有国王卑身下物，供养三宝。见有国王修斋行道、诵经礼忏，以度己身。见有国王正殿烧香，请大法师，布施持戒。。见有国王列百宝座，讲说大乘，昼夜不绝，敷扬妙义。见有国王亲率大臣来就观中，舍施金银珠玉、锦彩绞谷、车马宝饰、园池果林、衣服器物，种种供养以建福田。见有国王广度男女，以令出家。见有国王大度国人，绍隆正法。见有国王大立观宇，种种装严。见有国王广造法身，不惜财宝。见有国王抄写诸经，转读供养。见有国王旛华幢盖、香油上服，持用布施出家法身。见有国王终日讲说，赞扬玄化。见有国王奉持大戒，永无有犯。见有国王奉受三洞宝经，以佩於身。见有国王常发善心，专诚於道。见有国王舍位大臣，自身出家。见有国王放诸宫女，普令入道。见有国王散诸财宝，以与国中贫穷老病。见有国王谦卑下下，不损於物。见有国王形容端雅，智慧明远。见有国王年命长远#1。见有国王以道化人，人民欢乐。见有国王常行清浄，不色不欲。见有国王常行慈悲，不杀不害。见有国王常行施舍，不吝不惜。见有国王常行欢喜，不嗔不怒。见有国王常行俭约，不奢不泰。见有国王常行大慈，怜愍众生。见有国王常行精进，不退不转。见有国王常行清洁，不饮食酒肉、荤辛非法之食。见有国王手常抄经，广布一切。见有国王大散财物，布施法身。见有国王与诸道士日日讲论，咨问妙旨。见有国王日日斋会，供养道士。见有国王专心於道，求解脱法。见有国王至诚奉戒，求出世法。见有国王种种功德，以求神仙。见有国王放生求死，然灯烧香，广济一切。见有国王舍诸宫殿，置造灵观。见有国王诣诸名山，咨求上道。见有国王请诸大德尊者，忏悔受戒，以灭宿愆。
　　见有皇后独处深宫，专心於道。见有皇后舍弃六宫，归身入道。见有皇后舍身衣服、卧具、玩物，以用布施。见有皇后劝请国王，度人弘法。见有皇后咨请国王，请诸大德讲经论议。见有皇后大造经像，以度众生。见有皇后多送香华，就观供养。见有皇后持宝百物，以施出家法身。见有皇后奉持净戒，曾无亏犯。见有皇后不染六尘，身心清静，供养三宝。见有皇后劝请国王，令男女出家。见有皇后常发道心，广建功德。见有皇后常行布施，不惜身命。见有皇后常立百座，以讲大乘。见有皇后常以一心，遍修诸善。见有皇后弃舍王宫，入山学道。见有皇后请诸大德，受戒忏悔。见有皇后一心注念，佩奉宝经。见有皇后手营百味，供养三宝。见有皇后精进经像，烧香礼诵。见有皇后随其国王，生无量心。
　　见有王子随其国王，一一发心。见有王子身愿出家。见有王子不用荣华，布衣持法。见有王子专心於道，曾不懈怠。见有王子常诵真经。见有王子广建福田，为未来业缘。见有王子静持三业，身常清净。见有王子常与道士讲论妙义。见有王子诣诸道士，咨问经法。见有王子常於灵观，礼拜、烧香，布施、忏悔。
　　见有大臣随王发心，广建功德。见有大臣布施衣物，及以田宅、奴婢、牛马、车乘、卧具种种，以入出家法身。见有大臣舍其居宅，建立灵观。见有大臣广造经像，以为福田。见有大臣忠正智慧，助道兴化。见有大臣常生欢喜，乐闻经法。见有大臣志愿出家，绍隆大法。见有大臣身相齐整，果报具足。见有大臣立百宝座，屈请大德讲说三洞大乘，为国土人民开发善道。
　　见有百官生无量善心，随顺国王作种种功德。见有百官常运一心，为大法利。
　　见有道士常以经戒，劝说国王。见有道士常造经像，愿国安宁。见有道士教化人间，修营灵观、堂殿、楼阁，任力所堪，作当来众生归依窟宅。见有道士常行慈悲，愿度一切。见有道士常敷百座，讲说大乘，利人益国。见有道士精心於道，诵经持戒，昼夜不绝。见有道士广建福田，作诸桥梁。见有道士勤心於物，常说妙法，以导愚迷。见有道士常为众生忏悔受戒，普使归依。见有道士鸣钟打磬，六时不舍。见有道士独坐空山，吸景餐松。见有道士广行乞取，以散贫穷六疾。见有道士符禁汤药，救疗众生、一切病苦。见有道士堂堂无匹，作天人师。见有道士常在道场，抄经写像。见有道士独守空房，诵咏宝经，日不暂舍。见有道士智慧方便，自在无阂。见有道士欢喜一心，绍隆大法。见有道士每发慈心，开度一切。见有道士放赎生命，救济饥寒，布散衣食，大设福田。见有道士道边立井，起屋造堂，建诸桥梁，竹木华果，抚其劳苦。
　　见有女官端坐幽房，专心念道。见有女官烧香诵经，昼夜不怠。见有女官忏悔礼拜，以求升仙。见有女官精进不倦，愿得转身。见有女官发大善心，开度一切。见有女官大造经像，教化有缘。见有女官建立殿堂、台阁、楼榭、帐座、旛幢，供养天尊。见有女官修造玄坛、堂殿、廓宇，周圆具足，劝化当来。见有女官布施财物，无所吝惜。见有女官心无健羡，绝於人我。见有女官供事明师，曾不退惰。见有女官诵习经典，讲说大乘。见有女官断谷休粮，清浄身心。见有女官长斋奉戒，归诚於道。见有女官散施财物，济惠穷乏。见有女官放生赎命，普使快乐。见有女官作诸衣服，奉上法师大德。见有女官翘勤星汉，以求轻举。
　　见有贤者归心三宝，无有退转。见有贤者舍身命财，以用布施，供养三宝。见有贤者不惜妻子，供给出家法身。见有贤者烧香续命，礼拜供养。见有贤者舍家资财，田宅奴婢、器物车马、七宝名衣、种种器玩，供养三宝。见有贤者念念发心，常在於法。见有贤者修营灵观，不辞劳苦。见有贤者抄经铸像，罄竭家资。见有贤者修斋奉戒，常时不阙。见有贤者清净六根，不色不欲。见有贤者不偷不盗，常行布施。见有贤者心常观喜，无憎无爱。见有贤者精炼己身，不饮酒食肉，及以荤辛。见有贤者断发舍身，以用布施。见有贤者大小炼身，以求仙道。见有贤者割肉饴禽，慈悲於物。见有贤者形容端正，受命长远。见有贤者聪明智慧，悟解大乘。见有贤者忠孝君父，友悌骨肉。见有贤者父子和睦，六亲喜悦。见有贤者富贵巍巍，常用布施。
　　见有贤者六根具足，无有损伤。见有贤者受生快乐，无疾无痛，不饥不寒。见有贤者众人敬爱，所在致惜。见有贤者发大弘誓愿，舍身受身，常值经一法，受持供养。
　　见有女人清静身心，受持经法，礼拜供养。见有女人独坐幽室，不杂於世。见有女人专诚於道，以求度身。见有女人以身供养，无所吝惜。见有。女人随顺师尊，不生疑惑。见有女人舍身上服，随法所用。见有女人专诚慊款，精心奉法。见有女人造诸法服，以用布施。见有女人广将华果，供养三宝。见有女人常持妙戒，精进不退。见有女人清浄六根，无诸欲想。见有女人心行纯和，不染世事。
