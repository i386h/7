太上洞玄灵宝五符序
　　经名：太上洞玄灵宝五符序。撰人不详，约出於魏晋。系古《灵宝经》之一。葛洪《抱朴子内篇》已引此书。原本一卷，后分为三卷。底本出处：《正统道藏》洞玄部神符类。
　　太上洞玄灵宝五符序卷上
　　玄古淳和，阴阳无纶，风不动条，云雨以期。万物自纵其心，苍生任道而行，上对天心，天人合气，吉凶未兆，不闻杀害，无惧其凶，不欣其吉，不见可否之言，不生是非之教，淳然不亏，峙然不损，推天任性，委地纵命，千万获所，莫伤其和。其和不伤，天地无主，无主而自化，不教之大训，保其命也。于时人年九万岁，然后始学仙道而升天，斯将可谓至化，顺其心也，可谓至教。化之无方，教亦无迹，迹不见而功存，训长守而德没，可谓天和其象，三才比同者也。逮三皇之世，淳和欲缺，微有得失，笑怒於色，作制以定兆民，神灵营身以全众生，可谓竞始学而争已发。群称言得失，相自然诸兆，卦象万物，计玄虚於天地，有事之兴，於此兆矣。及轩辕之时，神农世衰，诸侯相侵，互有攻夺之气，明主弗能正也。於是黄帝习甩干戈，以征不廷，诸侯咸来宾从，而蚩尤最为暴，莫之能伐。炎帝欲侵凌诸侯，诸侯咸归轩辕，轩辕乃修德振兵，治艺五种，拯抚万民，度四方，教熊罴狼豹、貙虎玃天、以与炎帝战于阪泉之野，三战然后行其志。蚩尤作乱，不用帝命，於是黄帝乃征师，与蚩尤战于琢鹿之野，遂克之。诸侯因尊轩辕为天子，代神农氏，以云命为云师，置左右大监，监于万国鬼神，山川封，禅，与为多焉。获宝鼎，迎日推荚，举风后、力牧、恒先、大鸿以治民。顺天地之纪，幽明之故，死生之说，存亡之难，时播百谷草木，淳化鸟兽蚕桑，网罗日月星辰、水泥土石金玉，劳勤心力耳目，节用水火什物，有土德之瑞，故号黄帝。黄帝生二十五子，其得姓者十四人。黄帝居轩辕之丘，而娶西凌之女，是为累祖。累祖为黄帝正妃，生子二人，其后皆有天下。其一曰玄嚣，是为青阳，青阳降居江水。二曰昌意，降居弱水。昌意娶蜀山女，曰昌仆，生高阳。高阳有圣德，是为颛顼，黄帝之孙而昌意之子。渊淳有良谋，疏通而智，养财以任地，载时以象天，依神灵以信义，治气以教民，洁诚以祭祠，北至於幽陵，南至於丹穴，西至於流沙，东至於蟠木，动静之物，大小之神，日月所照，莫不称属焉。生子曰穷蝉，而玄嚣之孙高辛立，是为帝誉。帝誉，黄帝曾孙也。高辛父曰乔极，乔极父曰玄嚣，玄嚣父曰黄帝。自玄嚣与乔极，不得在位，至高辛即帝位，高辛於帝颛项为族子。高辛生而神灵，自言其名普施利，不於其身，聪以知远，明以察微，顺天之义，知民之急，仁而威，惠而信，修身而天下服。取地之物而节用之，抚教万民而利诲之，历日月而迎送之，明鬼神而敬事之，其色恢恢，其德巍巍，其动也时，其服也工。帝誉既执中而尸天下，日月所照，风雨所至，莫不从助，总得天地之心。其时有天人神真之官降之，乘宝盖玄车而御九龙，策云马而发天窗，自称九天真王、三天真皇，并执八光之节，佩景云之符，到于牧德之台，授帝誉以九天真灵经、三天真宝符。九天真金文，上开太上通天之气，下敷元生神祇之度，极混沌而为限，终地幽以生初，唯天真而造其涯，下贤不能达其际。九气所陶，神齐日月，於是物木名草，万品分别，甘苦有味，金石殊类。上以营天，使二仪无遗；下以营人，令年命无坠。达观者希其意，入妙者仰其味。然其文繁盛，天书难了，真人之言，既不可解，太上之心，众叵近测，自非上神启蒙，莫见发髴。是以帝喾自恨，其才下徒，贵其书而不知其向。帝喾乃祭天帝北河之坛，藏於钟山之峰，封以青玉之匮，以期后圣有功德者令施。其幽滞钟山，弱水之北一万九千里，山高万二千里，其上方七千里，周回三万里，自生千芝神草千四十种，人但脚履其上，三步乃仙矣。上有金台七宝紫阙，元气之所舍，天帝君所治处也。日月所不能照，钟山光耀，昼夜朗然，照明十万里外，皆星汉所不及。圣人言天缺无西北，是钟山照九阴之下矣。帝尧之时，洪水滔天，山坏陵崩，下民斯忧。尧命求能治水者，群臣四岳皆曰：鲧可。尧曰；鲧为人负命毁族，心有缺伤，恐不可用。四岳曰：未见复有贤於鲧者，愿帝试之。於是尧听四岳，用鲧治水，九年而水不息，功用不成。於是帝尧乃求人，更得舜。舜登用，摄行天子之政，巡狩行视，鲧之治水无功，乃殛鲧，於羽山以死，天下皆以舜之诛为是。於是舜举言鲧子禹能，而使续鲧之业。尧崩，帝舜问四岳：有能美成尧之事者，使居官。皆曰：伯禹为司空，可成业，美尧之功。舜曰：嗟然，命禹卒平水土。禹拜稽首，让於契、后稷、皋陶。舜曰：汝其往亲尔事。禹为人敏给聪济，其德不违，其仁可亲，其言可信，声为律，身为度，称以土，衅衅穆穆，为纲为纪。禹乃遂与益、后稷，奉帝命诸侯百姓、兴人徒以敷土，随山刊木，定高山大川。禹伤先人父鲧之功不成受诛，乃劳身焦思，居外一十三年，过家门不敢入，薄衣食致孝於鬼神，卑宫室尽力於沟洫，陆行乘车，水行乘船，泥行乘橇，山行乘标，左准绳，右规矩，戴四时以开九州，通九道，陂九泽，度九山。令益与众庶，稻可种卑湿。令后稷与众庶，艰食鲜食，食少，调有余相给，以均诸侯。禹乃行相地宜所有以贡，及山川之便利。禹行自冀州始，既为壶口断梁及岐，既修太原至于岳阳，覃怀致功至於衡漳恒卫，既从大陆，既作岛夷皮服，夹右碣石入于河，济河衮州。九河既导，雷夏既泽，浓沮会同，桑土既蚕，於是民得下丘居。后土平济，大水既消，尔乃巡狩於钟山，祀上帝於玉阙，归洪勋於天后，还大成於万灵。然后登彼玄峰，於绣岭之阿，琼境之上，忽得此书。禹乃更恭斋罄林幽岫，请奉佩身。真人告禹曰：汝功德感灵，天人并助，而年命向雕，崄矣哉。乃口诀以长生之道，示以真宝服御之方，分擿而别。还乃计功劳於会稽之野，召会群神於东越之山。是时四气温隆，清风既鼓，玄功妙畅，虚心内治。思退身以灭迹，惟藏景於幽绪，睹九天之灵奥，观三天之宝囿，动以全生为大顺，静以舍精为久视，上览天经，幽而难缕，下愍群生，不知其纪，乃复凤翔南山，龙峙海岛，更撰真灵之玄要，集天宫之宝书，差次品第，分别所修，行五色定其方面，名其帝号。太上本名为灵宝五符天文，藏於玄台之中，坚石之碽古红切，隐於苗山之岫，万年一出，以示不朽。其一通书以南和丹缯，隐斋於蒙笼之丘讫，封以金英之函，印以玄都之章，命川泽水神以付震水洞室之君，须三千之会，当传与水师傅伯长。其石碽之文，乃待大劫一至而宣之耳。禹於是服灵方以匿景，葬兹山以显终，仙人解其契，愚人谓其亡。大哉神化，委身归宗，於戏大君，玄迹高通，万劫验其书，道咏须其方，阳九见上天之书，百六告传氏之功，洪唱千代，兰气遗芳者也。
